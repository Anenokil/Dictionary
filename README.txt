Программа-менеджер словарей для запоминания иностранных слов и увеличения словарного запаса.

================================= Возможности программы =================================
---------------------------------------- Словари ----------------------------------------
Вы можете создать неограниченное количество словарей.

Благодаря этому:
- Вы можете изучать несколько языков одновременно.
- Программой могут пользоваться несколько людей.

-------------------------------------- С т а т ь и --------------------------------------
Вы можете добавить в словарь неограниченное число статей.
Каждая статья должна содержать одно слово и хотя бы один его перевод.

Помимо этого:
- Можно добавить в статью несколько переводов.
- Можно добавить в статью сноски (примечания, подсказки, что угодно).
- Можно добавить в статью формы слова, например множественное число, падежи и что угодно ещё (см. Универсальность).
- Можно добавить статью в избранное.
  В словаре у таких статей слева будет пометка (*).
- Можно создать несколько статей с одинаковым словом.

------------------------------------- Изучение слов -------------------------------------
Чтобы посмотреть статьи, которые вы добавили, можете напечать словарь.

Также вы можете учить добавленные слова.
- Есть несколько режимов учёбы.
- Если вы неверно ответите, то вам будет предложено добавить статью в избранное.

------------------------------------ Универсальность ------------------------------------
Программа подходит для изучения любого языка:
- Помимо таких грамматических категорий, как падеж, число или род, вы можете добавить любые другие.
  Для этого в программе перейдите в Настройки -> Настройки словаря -> Грамматические категории.
  Для каждого словаря используются отдельные категории.
- Если у вас отсутствует необходимая раскладка клавиатуры, вы можете использовать специальные комбинации.
  Для этого в программе перейдите в Настройки -> Настройки словаря -> Специальные комбинации.
  Пример: #A -> Ä

=============================== Пример статьи  из словаря ===============================
(*) [2;  75%] house: дом, здание
              > сноска_1
              > сноска_2
-----------------------------------------------------------------------------------------
Слева от каждого слова в квадратных скобках кратко записана ваша статистика:
количество последних неверных ответов с этим словом и процент верных ответов с этим словом.
